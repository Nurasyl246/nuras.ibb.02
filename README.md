## Фотохостинг

Этот сайт является простым фотохостингом. Вы можете загружать изображения, которые будут храниться в облаке и отображаться в галерее.

### Как использовать:
1. Откройте index.html в браузере.
2. Выберите изображение(я) с помощью кнопки "Выбрать файл".
3. Нажмите "Загрузить" для отправки на сервер.
4. Изображения появятся в галерее.

### Настройка:
- Для загрузки изображений используется API imgbb.com.
- Вам нужно получить API ключ на https://api.imgbb.com/ и заменить 'YOUR_API_KEY' в index.html на ваш ключ.
